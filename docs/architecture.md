# TorViNet 模型架构详解

## 论文信息

**TorViNet: A Spatiotemporal Deep Learning Network for Tornado Detection in User-Captured Social Media Videos**

- 作者：Hongjin Chen, Kanghui Zhou, Zhonghua Zheng, Lei Han, Yongguang Zheng
- 期刊：Expert Systems With Applications, 2026
- DOI：10.1016/j.eswa.2026.000007

## 核心创新

TorViNet 针对社交媒体视频中龙卷风检测的三大挑战提出了解决方案：

### 1. 冗余帧问题
**问题**：用户拍摄的视频中存在大量无信息的冗余帧（如静止背景、模糊画面）

**解决方案**：**DFSM（Dynamic Frame-level Selection Module）**
- 动态选择最具代表性的帧
- 过滤时间上无信息的内容
- 提高计算效率，减少噪声

### 2. 弱小目标问题
**问题**：龙卷风在远距离拍摄时呈现为小目标，细节特征易丢失

**解决方案**：**SFMHA（Spatial-Frequency Multi-Head Attention）**
- 结合空间域与频率域特征
- 增强细粒度涡旋结构感知
- 多头注意力机制捕捉多尺度特征

### 3. 动态遮挡问题
**问题**：云层、雨幕、建筑物等动态遮挡龙卷风视觉特征

**解决方案**：**LC-MLP（Local Contrast MLP）**
- 对比感知细化模块
- 抑制背景干扰
- 突出龙卷风目标

## 整体架构

```
输入视频 (B, 3, T, 224, 224)
    ↓
┌─────────────────────────────────┐
│  DFSM 动态帧选择                 │
│  - 计算帧级重要性权重             │
│  - 选择关键帧                    │
│  输出: (B, 3, K, 224, 224)      │
└─────────────────────────────────┘
    ↓
┌─────────────────────────────────┐
│  SFMHA 空间频率注意力             │
│  - 空间特征提取                  │
│  - 频率域增强                    │
│  - 多头注意力融合                │
│  输出: (B, C, K, H, W)          │
└─────────────────────────────────┘
    ↓
┌─────────────────────────────────┐
│  LC-MLP 对比感知细化              │
│  - 局部对比学习                  │
│  - 背景抑制                      │
│  - 目标增强                      │
│  输出: (B, C, K, H, W)          │
└─────────────────────────────────┘
    ↓
┌─────────────────────────────────┐
│  分类头                          │
│  - 全局平均池化                  │
│  - 全连接层                      │
│  输出: (B, 1) [龙卷风概率]       │
└─────────────────────────────────┘
```

## 模块详解

### DFSM - 动态帧选择模块

**功能**：从 T 帧中智能选择 K 个关键帧

**实现方式**：
1. 计算每帧的时间重要性权重
2. 使用 SE-Block 进行通道注意力计算
3. 选择权重最高的 K 帧

**优势**：
- 减少冗余计算
- 聚焦关键时刻
- 提高信噪比

### SFMHA - 空间频率多头注意力

**功能**：在空间和频率域同时增强特征

**实现方式**：
1. **空间分支**：标准 3D CNN 特征提取
2. **频率分支**：Fourier 变换 + 频率域滤波
3. **融合**：多头注意力机制融合两个分支

**优势**：
- 捕捉空间结构（涡旋形状）
- 捕捉频率特征（旋转运动）
- 多尺度特征融合

### LC-MLP - 局部对比细化

**功能**：通过对比学习抑制背景，突出龙卷风

**实现方式**：
1. 计算局部特征的对比度
2. 增强高对比度区域（龙卷风）
3. 抑制低对比度区域（背景）

**优势**：
- 鲁棒性强
- 适应各种天气条件
- 减少误检

## 性能指标

### 数据集
- **规模**：10,000+ 条视频
- **来源**：社交媒体（抖音、Twitter 等）
- **标签**：龙卷风 / 非龙卷风

### 结果
| 指标 | 数值 |
|------|------|
| Accuracy | 91% |
| Precision | 0.92 |
| Recall | 0.87 |
| F1-score | 0.89 |

### 鲁棒性测试
- ✅ 嘈杂视频：准确率 88%
- ✅ 不稳定拍摄：准确率 89%
- ✅ 远距离拍摄：准确率 86%
